Getting Started

This application helps you to integrate your datasets with a dataset on Data science Platforma and visualize them on the map.

  1. Click on GD Wizard and study GD Wizard guideline and upload CSV transformation.
  2. No configuration is needed.
  3. To publish your dataset, you need to login in Platform Linked Data Nederland (PLDN) and create a token.
  4. Click on user settings.
  5. Click on API Token.
  6. Click on create token.
  7. Select all access level items.
  8. Select a token name and click on create.
  9. Now, you can copy the API token and paste the API in publish step to recognize your account and publish your dataset in the PLDN.
  10. Select a name for your dataset and click on the publish bottom.
  11. Click on services and create new services.
  12. Create new SPARQL service and select a name for your service.
  13. Remove limit 10 and save the default query.
  14. Click on three points front of your query name and click on edit.
  15. Configurate the query to public and update query.
  16. Copy the API address below the API variables.
  17. In put your API address in search bar and click on Go.
  18. Left click on markers to zoom in.
  19. Shift + left mouse double click is zoom out and box zoom in.
  20. Right click on building should display Knowledge Graphs datasets in a pop-up.
  21. Click at the top right to switch to aerial photos and Open Street Map.
  22. Click on blue links in pop-up to direct you to websites.
